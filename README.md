# bucket_sort_tdd

## Run tests
1) Clone this project and go in it's folder
```shell
git clone https://github.com/GeoffreyBrunet/bucket_sort_tdd
```
and go in it's folder
```shell
cd bucket_sort_tdd
```
2) Download pytest
```shell
python3 -m pip install pytest
```
3) Run tests with pytest
```shell
pytest
```
